# Hint 1 — direction

These are four independent one-liners, not one pipeline — solve and print
each `===Qn===` block separately. `rg` is for content (what's inside
files); `fd` is for paths (which files exist). Q1 and Q3 are content
questions; Q2 and Q4 are path/existence questions — reach for the right
tool for each rather than forcing everything through one of them.
