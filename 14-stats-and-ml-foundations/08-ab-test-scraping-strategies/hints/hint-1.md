Strategy B extracted the field on more attempts than strategy A did, in
this particular batch of 1500-vs-1500 pages. That alone doesn't tell you
much. If you ran the exact same bake-off again with a fresh batch of pages
-- same two strategies, same true underlying reliability -- you would NOT
get exactly the same proportions. Any finite sample wobbles around the true
rate just from which pages happened to land in it.

So the real question isn't "which strategy did better on this sample" --
you can answer that with a mean and a comparison operator, no statistics
required. The real question is: "if A and B actually had the SAME true
success rate, how surprising would a gap this large be, purely from
sampling?" If a gap this big (or bigger) would happen often just by chance,
you haven't learned much. If it would almost never happen by chance, that's
evidence something real is going on.

A hypothesis test is a way of turning "how surprising is this gap under the
assumption there's no real difference" into an actual number you can put a
threshold on. Start there before worrying about the specific formula.
